# java_labs

Лабораторні роботи з предмету **Об'єктно-орієнтований аналіз та проектування інформаційних систем**

## Система "Факультатив"
Існує перелік курсів, за кожним з яких закріплений один викладач. Студент записується на один або кілька курсів, дані про реєстрацію зберігаються. Після закінчення навчання викладач виставляє студенту оцінку, яка зберігається в архіві.


- **Teacher** — викладач, закріплений за курсом
- **Course** — курс із назвою, описом та тривалістю
- **Student** — студент, який записується на курси
- **Registration** — дані про запис студента на курс
- **Archive** — оцінка студента після завершення курсу
